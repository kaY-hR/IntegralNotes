# 2026-04-29 会話まとめ

## 目的

解析 block の output slot 初期 file 名を、従来の `slot名 + extension` から `slot名 + 英数字3桁 + extension` に変更する。

## 実装内容

- `src/shared/integral.ts` に `A-Z0-9` の3桁 suffix を生成する helper を追加した。
- 新規 Integral block 生成時の output path を `/Data/{slot名}{英数字3桁}{extension}` にした。
- main process 側で output が未指定または空の場合も、実行前補完時に同じ命名規則を使うようにした。
- output path field の placeholder は `/Data/{slot名}A1B{extension}` 形式で表示し、未保存 block の表示中に random 値が揺れないようにした。
- 要求文書と Python 実行設計文書へ、output 初期 file 名規則を追記した。
