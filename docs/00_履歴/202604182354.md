## 概要

- process chain viewer を optional な internal workspace-tool plugin として実装した
- Activity Bar から main workspace tab を開き、現在の note / file を起点に block / file chain を可視化できるようにした
- 関連する要求文書、設計文書、Issue を current 仕様へ更新した

## 実装

- `src/renderer/workspaceToolPlugins.tsx`
  - optional な workspace-tool plugin registry を追加
  - process chain viewer を app core 直書きではなく plugin 項目として登録
- `src/renderer/App.tsx`
  - Activity Bar item から workspace-tool plugin tab を開く導線を追加
  - plugin tab が active な間も、最後に focus していた通常 tab を context として渡すようにした
  - open 中の Markdown note 内容を override として viewer 側へ渡すようにした
- `src/renderer/ActivityBar.tsx`
  - sidebar view 以外の item も active 状態を個別に持てるようにした
- `src/renderer/ProcessChainViewer.tsx`
  - note 群から Integral block を抽出し、input / output / `createdByBlockId` を元に graph を構築する viewer を追加
  - block click で `note-path#BLK-...`、file click で既存 open 導線を使う read-only viewer にした
- `src/renderer/styles.css`
  - process chain viewer の graph UI、empty / loading / error state を追加
