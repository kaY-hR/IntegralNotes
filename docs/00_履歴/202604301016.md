# 2026-04-30 10:16 AI Chat skill 補完 popup

## 目的

AI Chat panel と inline AI popup で、`$` による明示 skill 指定を手入力だけでなく候補 popup から選べるようにする。

## 対応内容

- skill 指定記法を `$skill-name` に寄せ、空白または行頭の `$` だけを補完・認識対象にした。
- `src/shared/aiChatSkills.ts` に、textarea のカーソル位置から `$` trigger を検出する helper と skill 候補 filter を追加した。
- `src/renderer/AiSkillCompletionList.tsx` を追加し、skill 候補の popup list を共通 UI として実装した。
- AI Chat panel の composer textarea で `$` 入力時に skill 候補を表示し、`↑/↓`、`Enter`、`Tab`、マウス選択で `$skill-name ` を挿入できるようにした。
- inline AI popup の textarea でも同じ skill 候補 popup と選択操作に対応した。
- 既存の skill chip 表示と送信時の skill metadata 連携は維持した。
