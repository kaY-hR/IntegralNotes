# 2026-05-01 10:31 Playwright MCP

## 背景

- Codex が IntegralNotes を自分で起動し、画面操作を通じて動作確認できるようにしたい。
- 既存の `integral-analysis` MCP は別ソフトなので使わない。
- IntegralNotes アプリ内の AI Chat MCP client registry はまだ未実装であり、今回の対象ではない。

## 対応

- `src/dev/integral-playwright-mcp.mjs` を追加し、stdio MCP server として Playwright 経由で Electron を操作できるようにした。
- `package.json` に `npm run mcp:playwright` を追加した。
- `@modelcontextprotocol/sdk` と `playwright-core` を dev dependency として追加した。
- `src/main/main.ts` に `INTEGRALNOTES_USER_DATA_DIR` hook を追加し、自動確認用 Electron userData を通常利用と分離できるようにした。
- `docs/30_設計/95_PlaywrightMCP.md` に目的、起動方法、Codex 側 MCP 設定例、tools 一覧を記録した。

## 提供 tool

- `integral_status`
- `integral_snapshot`
- `integral_click`
- `integral_fill`
- `integral_press`
- `integral_wait_for`

- `integral_close`
