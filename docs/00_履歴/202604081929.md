# 2026-04-08 19:29 作業ログ

## 依頼

- 起動した画面に左サイドバーも上部メニューも見えない
- 想定していた新 UI ではなく、古い見た目に見える

## 原因推定

- ビルド出力内には `Research Notes Dock` や `Open Folder` を含む新 UI が入っていた
- 一方で `src/dev/start-renderer.cjs` の既存ロジックは、`127.0.0.1:5173` で HTTP 応答さえ返れば何でも「再利用可能な Vite サーバー」と見なしていた
- そのため、別プロジェクトや別セッションの 5173 サーバーを誤って再利用し、現在の IntegralNotes の renderer ではない画面を表示していた可能性が高い

## 対応

- `src/dev/start-renderer.cjs`
  - `GET /src/renderer/App.tsx` を取りに行くチェックを追加
  - その内容に `Research Notes Dock` が含まれる場合だけ、現在のプロジェクトの renderer とみなして再利用するように変更
  - 一致しない場合は
    - `Port 5173 is serving another app...`
    - と明示して終了するように変更

## 検証

- `node --check src/dev/start-renderer.cjs` 成功
- 既存の `dist` 出力内には新 UI の文字列が含まれることを確認済み

## メモ

- これで、別アプリの Vite サーバーを誤って掴んで「サイドバーがない」状態になる可能性を下げた
- 実際の反映確認は、既存の 5173 / Electron プロセスを閉じてから `npm run dev` を再実行するのが確実
