## 判断

- `plugins/*` は third-party plugin の最終置き場ではなく、bundled plugin の開発・検証用ワークスペースとして扱う。
- third-party plugin は将来的に別 repo でも同じ artifact 構成を作れればよい。
- host 実行は今回は `stdio executable` ではなく `CommonJS module load` の MVP とした。
