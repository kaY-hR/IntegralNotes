# 2026-04-18 Summary

## 依頼

- `ノート` button が hover しない限り見えず、通常時は白背景白文字に見える
- output 名 input を、薄い枠または非常に薄いグレー背景で入力欄らしく見せたい

## 対応

- `src/renderer/styles.css`
  - output 名 input の border を少し見える色へ寄せ、背景を淡いグレーにした
  - hover / focus 時は白背景に戻し、border も少し強めるようにした
  - `ノート` button は `.editor-surface .milkdown .integral-slot-row__link--note` を含む selector にして specificity を上げ、通常時も青背景白文字になるよう固定した
  - hover / focus-visible でも同系色のまま見えるようにした

## 検証

- `npm run build` が通った
