# 2026-04-29 21:24 AIメッセージのMarkdown表示

## 目的

AI Chatパネルとinline AIポップアップで、AI応答内の `**` などのMarkdown記法がそのまま見えてしまうため、通常メッセージをMarkdownとして表示する。

## 対応内容

- `src/renderer/AiMarkdown.tsx` を追加し、React要素として描画する軽量Markdownレンダラーを実装した。
- AI Chatパネルのユーザー/アシスタント通常メッセージと、streaming中のアシスタント応答をMarkdown表示に変更した。
- inline AIポップアップのユーザー/アシスタント通常メッセージと、streaming中のアシスタント応答をMarkdown表示に変更した。
- ツール実行ログや診断メッセージは、改行や空白を保つため従来どおりplain text表示のままにした。
- Markdown用のスタイルを `src/renderer/styles.css` に追加した。

## 確認

- `npm run build` が成功した。
- 既存のVite CJS API非推奨警告とchunk size警告は引き続き表示されるが、今回の変更によるビルドエラーはない。
