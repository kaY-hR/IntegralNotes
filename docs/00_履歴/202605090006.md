# 2026-05-09 00:06 Explorer sticky folder visual adjustment

## 依頼

- Explorer の sticky folder 行の上側にわずかな隙間があり、後ろの項目が見える。
- 展開中 folder に影または下枠線のような表示が見えるため消したい。

## 対応

- `FileTree` の sticky row 積み上げ間隔を実際の行高 `24px` に合わせた。
  - これまで `25px` だったため、複数階層の sticky folder 間に 1px の隙間が出ていた。
- sticky folder 行の `box-shadow` を削除し、下枠線に見える表示を消した。
- sticky folder 行は角丸を外し、背後の行が角から見えにくい矩形表示にした。
- 追加確認で、sticky folder の上側に見える約 8〜10px の隙間は `sidebar__tree` の上 padding が sticky の移動上限として残っていることが原因と判断した。
- Explorer tree の通常時の上余白は維持しつつ、sticky folder がスクロール領域上端まで上がれるよう、上 padding を scroll container から `file-tree` 側へ移した。

## 確認

- `npm run build` 成功。
