- `@@` は action picker、`@@name` は直接 action 起動、`??` は新仕様では `@@continue` alias、`>>` は `@@mkpy` alias とした。
- skill / `$$` trigger は MVP 対象外にした。

- 最終 commit tool は `insertMarkdownAtCursor` に統一し、Python block は `createPythonBlockDraft` で draft を作ってから Markdown として挿入する方針にした。

- renderer の inline AI popup を `submitInlineAction` 経由に統一した。

- Inline Action 管理 UI を追加し、GUI から action の作成・編集・削除ができるようにした。
- workspace template の `Notes/.inline-action/` に標準 action 定義を追加した。
- `.inline-action` は file tree / search / AI workspace snapshot / managed data auto-register 対象から除外した。

## 追補2

- `.inline-action` が workspace root 直下に展開されているが、アプリの file tree では隠しフォルダ表示ONでも見えないことが分かった。
- 原因は workspace snapshot 生成時に `.inline-action` を無条件除外していたこと。
- `.inline-action` は通常は非表示でよいが、隠しフォルダ表示ONなら見えるべきなので、snapshot の無条件除外から外した。
- search / managed data auto-register / AI workspace snapshot からの除外は維持した。

## 追補3

- `@@` picker の操作感を `>` / `[` / `![` の editor completion に合わせた。
- `@@` 入力中は inline AI form popup を出さず、editor 上の completion list として候補を出すようにした。
- 続けて入力した query で action name を前方一致 filter する。
- 上下キーで選択し、Enter / Tab または click で action を起動する。
- action name を手打ちで最後まで入力し、候補が一意に確定した場合も同じ action を起動する。

## 追補: inline action commit tool の terminal 化

- `insertMarkdownAtCursor` / `insertPythonBlock` を呼んだ時点で task agent loop を終了するようにした。
- これにより commit tool 実行後にモデルの最終応答を追加で待ち続け、追記タイミングで hang して見える問題を抑制した。
- Inline Action (`@@`) だけでなく、旧 inline insertion / python block / promptless continuation 経路にも同じ terminal tool 指定を適用した。

## 追補: inline popup のキャンセル操作

- inline popup の `[x]` は submit 中でもキャンセルとして動作するようにした。
- close button の pointer down で popup drag へイベントが流れないようにし、その場で popup を閉じる。
- cancel 後に遅れて inline action の結果や error が返っても、stream id guard により popup 復活や挿入を行わない。

## 追補: AI Chat Settings Dialog の整理

- AI Chat Settings Dialog から旧 inline AI (`??` / `>>` / `@@`) 用 system prompt 編集欄を削除した。
- Dialog 内で編集できる prompt は AI Chat panel 本体の system prompt のみにした。
- Reset Prompt は chat panel prompt だけを既定値に戻し、Inline Action 側の prompt 設定は専用UIで管理する方針にした。
- モデル/API key/shell path の設定グリッドを AI Chat Settings Dialog 内では 1 列表示にして、横並びで詰まらないようにした。
