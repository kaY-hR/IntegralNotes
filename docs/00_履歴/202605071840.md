# 2026-05-07 18:40 AI 画像 tool の thumbnail / crop 化

## 依頼

- `main_ws` 側にも、AI tool が画像を model に渡すときは既定で 500px ちょいに小さくする方針を入れたい。
- 必要に応じて crop で領域指定して渡せるようにしたい。

## 対応

- `readWorkspaceImage` の model 出力を、既定で長辺 512px 程度・120KB 目安の JPEG thumbnail に変更した。
- `renderWorkspaceDocument` の screenshot model 出力も同様に thumbnail 化した。
- 両 tool に `crop: { x, y, width, height }` を追加した。
  - `readWorkspaceImage` は source image pixel。
  - `renderWorkspaceDocument` は rendered screenshot pixel。
- tool instruction と trace summary に、thumbnail / crop の情報が出るようにした。
- `docs/30_設計/86_AIChatPanelとAgentTooling.md` に仕様を追記した。

- `npm run build` は renderer build 後、main TypeScript で未使用引数 1 件により失敗した。
- 未使用引数を整理後、`npm run build:main` が成功した。
- 最終確認として `npm run build` が成功した。
