## 補足

- `blob:` URL は renderer process に依存する一時 URL であり、app 再起動後に source 文字列だけから復元することはできない。そのため、修正ポイントは note save 時ではなく upload 発生時と整理した。
- この文書化がひと段落したら、本題の `workspace file` Markdown link 実装へ戻る方針で合意した。
- 今回は文書整備と Issue 更新のみで、コード実装や build / test はまだ行っていない。
