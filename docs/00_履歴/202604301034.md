# 2026-04-30 10:34 AI Markdown テーブル表示対応

## 目的

AI Chat パネルと inline AI ポップアップで、AI 応答内の Markdown table が通常のテキストとして表示されるため、表として描画できるようにする。

## 対応内容

- `src/renderer/AiMarkdown.tsx` の軽量 Markdown renderer に table block を追加した。
- `| Header |` + `| --- |` 形式の Markdown table を検出し、`thead` / `tbody` を持つ table として描画するようにした。
- `:---` / `---:` / `:---:` による left / right / center alignment に対応した。
- table cell 内でも既存の inline Markdown (`**bold**`, `*italic*`, `` `code` ``, link) を描画するようにした。
- `src/renderer/styles.css` に AI Markdown table 用のスタイルを追加し、横幅が足りない場合は横スクロールできるようにした。

## 変更ファイル

- `src/renderer/AiMarkdown.tsx`
- `src/renderer/styles.css`

## 確認

- `npm run build` が成功した。
- `git diff --check` は改行コード warning のみで、whitespace error は出なかった。
- Vite の CJS Node API deprecation warning と chunk size warning は既存 warning として引き続き表示された。
