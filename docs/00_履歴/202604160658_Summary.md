## 会話要約

- embed block について、画像は元サイズ以上に拡大しないようにしたい、という追加要望があった。
- あわせて、前回入れた縦方向 resize の高さを再オープン後も保持したい、という依頼があった。

## 実装

- `src/shared/workspaceLinks.ts`
  - workspace target の末尾に付ける embed 高さ metadata として `#integral-embed-height=NNN` を扱えるようにした。
  - path 解決時はこの suffix を無視し、rename / move rewrite 時には suffix を保持したまま path 部分だけ書き換えるようにした。
- `src/renderer/workspaceEmbedFeature.tsx`
  - embed source から永続化された高さ metadata を読み、初期表示時に復元するようにした。
  - resize 完了時は default 高さとの差分を見て、必要な場合だけ `#integral-embed-height=NNN` を source に書き戻すようにした。
  - default 高さに戻した場合は metadata を消し、source をなるべく素の target に戻すようにした。
- `src/renderer/styles.css`
  - image embed は `object-fit: scale-down` に変更し、枠が広くても元画像サイズ以上には拡大しないようにした。

## 確認

- `npm run build` : 通過

## 補足

- 高さの永続化は workspace target ベースの embed を対象にしている。
