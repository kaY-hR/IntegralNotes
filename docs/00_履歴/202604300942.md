# 2026-04-30 09:42 inline AI popup の行頭トリガーとドラッグ移動

## 目的

`??` / `>>` の inline AI popup が文中でも発動し得るため、`>` と同様に行頭でのみ発動するようにする。あわせて popup の上部枠を掴んで移動できるようにする。

## 対応内容

- `src/renderer/MilkdownEditor.tsx` の inline AI trigger 判定を、現在行の内容が `??` または `>>` だけの場合に限定した。
- inline AI popup のヘッダーに pointer drag 処理を追加し、viewport 内に収まる範囲で `left/top` を更新できるようにした。
- popup 幅の実 CSS に合わせて初期表示位置の clamp 幅を `520px` に揃えた。
- `src/renderer/styles.css` で popup header に `grab` / `grabbing` cursor、`touch-action: none`、`user-select: none` を追加した。

## 確認

- `npm run build` が成功した。
- Vite の CJS Node API deprecation warning と chunk size warning は既存の build warning として表示された。
