## フェーズ

jsonを使った独自UIの挿入

## 要求

とりあえず、スラッシュコマンドから"Integral"のセクションにあるメニューを選ぶと、下記の様なjsonを挿入することはできる

```json
{
    "type":"LC.Method.Gradient",
    "params":{
        "analysis-time":8,
        "time-prog":[{"time":0,"Conc":10},{"time":8, "Conc":100}]
    }
}
```

そこで今度は、こういったjsonを元に独自UIを挿入するような仕組みを仮実装したい。

## 補足

### Plugin仕様

こういった独自UIはPluginとして拡張可能なようにすべきだが、そのあたりの検討は全くできていない。
jsonのスキーマも、どういうルールにするのかも決まっていない。
下記、適宜提案してほしい。

* pluginの実行ファイルはどこにインストールされるべきか
* pluginが満たすべき制約(インターフェース)は何か
* どうやってIntegralNotes本体がインストール済みpluginを認識するのか

などなど。

**ただし、一旦はpluginのことは考えずに、jsonを独自UIに置換する、という検討にフォーカスしてOK**。余力があれば、pluginシステムについて仕様を検討し、文書化してほしい。
