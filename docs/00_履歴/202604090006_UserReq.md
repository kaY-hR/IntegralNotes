## フェーズ

jsonを使った独自UIの挿入機能の検討

## 要求

まずは、何かしらのtriggerで項目を選択 -> 選ぶとjsonが挿入される を実装したい。

例えば、
* '>>'と打つ
* スラッシュコマンドメニューのようなメニューが出る。
* そこで例えば`LCのグラジエント設定`のようなメニューを選ぶ
* イメージとしては、例えば下記の様なjsonが挿入される
```json
{
    "type":"LC.Method.Gradient",
    "params":{
        "analysis-time":8,
        "time-prog":[{"time":0,"Conc":10},{"time":8, "Conc":100}]
    }
}
```
* 本当は表示上は独自のhtmlを表示できるようにしたいが、一旦はjson挿入のみ。

## 補足

milkdown crepeでは、どうもスラッシュコマンドに新しくメニューを追加するのが難しいらしい(要確認)
そこで、スラッシュコマンドとは別のコマンドをtriggerとして新しく追加することで(例えば'>'など)、別メニューを作れるかもしれない

あと、そもそも独自UIとしてhtmlは挿入可能なのか？は引き続き調査する必要がある。

## Plugin仕様

こういった独自UIはPluginとして拡張可能なようにすべきだが、そのあたりの検討は全くできていない。
jsonのスキーマも、どういうルールにするのかも決まっていない。
適宜提案してほしい。
