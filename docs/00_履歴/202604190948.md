## 概要

- process chain viewer 上で node を click したとき、target tab を background open して viewer 自身の focus を維持するようにした
- 「node click で画面がバッと切り替わる」違和感を減らすための UX 調整

## 実装

- `src/renderer/App.tsx`
  - `OpenWorkspaceFileOptions` に `preserveFocus` を追加
  - `openWorkspaceFile()` で `preserveFocus = true` のときは `selectTab()` を行わず、既存 tab / 新規 tab のどちらも background open するようにした
  - `FlexLayout.Actions.addNode(..., select)` の `select` を `preserveFocus` と連動させた
  - process chain viewer からの `onOpenWorkspaceFile` / `onOpenWorkspaceTarget` は `preserveFocus: true` を使うようにした
  - `handleLayoutModelChange()` の `ADD_NODE` 処理は、追加された tab id ではなく実際に選択されている tab id を読むようにした
