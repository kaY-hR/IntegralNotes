# 2026-05-04 00:05 Top bar menu 整理

## 依頼

- top bar の menu を整理したい。
- 一般的な `[ファイル] [設定]` のような分類にまとめたい。

## 対応

- `src/renderer/App.tsx` の top bar 直下に並んでいた操作ボタンを dropdown menu 化した。
- menu 構成は次の分類にした。
  - `ファイル`: フォルダを開く、VSCodeで開く、初期化/更新、データ登録
  - `ツール`: Plugins
  - `設定`: 本体設定、Inline Actions
- menu 外クリック、Escape、dialog 表示時に menu を閉じるようにした。
- `src/renderer/styles.css` に app menubar dropdown 用のスタイルを追加した。

## 確認

- `npm run build` 成功。
