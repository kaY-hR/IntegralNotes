## 実施内容

- `docs/Issues.md`
  - `## [ ] 8. markdown 以外の text 系ファイルも編集・保存したい` を追加。
  - 内容として、
    - `json / txt / csv / log / xml / yaml / py / html / svg` などの text 系ファイルは plain text editor で編集・保存したい
    - `markdown` は Milkdown 維持
    - `html / svg` は preview / source 切替が望ましい
    - image / binary は preview / read-only のままでよい
    を記録した。
