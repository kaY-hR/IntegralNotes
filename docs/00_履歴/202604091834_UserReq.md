## フィードバック

### 全体

#### 表示の拡大縮小
優先重み:10

* ctrl+'-'で表示縮小できるのに、ctrl+'+'で拡大が出来ないため、一回縮小するとそれ以上拡大できなくなる
* ピンチアウトやホイールズームでも拡縮ができると望ましい

#### リリース版はelectronのデフォルトメニューを消したい
優先重み:1

electronのデフォルトトップバー(File/Edit/View/Window/Helpなどがあり、Toggle Dev Toolsなどが選択できる)については、npm run devの時は出したいが、インストーラでインストールされたものについては消したい。

でも、技術的に難しいなら保留でOK

### エクスプローラー

#### アイテム選択を外せない
優先重み:10

* アイテムがないところをクリックしても、最後に選択したアイテムが選択状態になる
* そのため、root直下にファイルやフォルダが作れない

#### root直下にファイルやフォルダが作れない
優先重み:10

アイテムがないところを右クリックしたときに、ファイルやフォルダ追加のコンテキストメニューを表示してほしい。

### ファイル編集

#### タブ削除時の動作
優先重み:5

ファイルタブを全部消すと、即座に直前に開いていたファイルが勝手に表示される。凄く違和感があるからやめたい。

#### スラッシュコマンドメニューが残る
優先重み:2

スラッシュを打った状態でスラッシュコマンドメニューを開くと、その後エディタ内で他の箇所にカーソルを移さない限り、他のGUI部品をクリックしたりしてもスラッシュコマンドメニューが最前面に表示され続ける。可能なら、エディタ以外の部分にフォーカスを移した時点でスラッシュコマンドメニューは非表示にしたい、、が、milkdownの仕様ならちょっと対応は要検討

#### テキスト直編集
優先重み:1

可能なら、wiswigモード以外に、生のマークダウンファイルを編集できるモードに切り替えられると嬉しい
