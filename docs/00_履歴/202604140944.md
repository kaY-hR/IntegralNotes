## 会話要約

- queue の残件として、main app 側でも explorer で `.md` 以外を見せたい、かつノートタブでも `chunk viewer` 相当の表示をしたい依頼があった。

## 実施内容

- explorer の `.md` 限定を解除した。
  - `src/main/workspaceService.ts`
    - `isVisibleEntry()` を更新し、`.` 始まり以外のファイルをすべて tree に出すようにした。
- main process に generic file read API を追加した。
  - `src/shared/workspace.ts`
    - `WorkspaceFileViewKind`
    - `WorkspaceFileDocument`
    - `readWorkspaceFile()`
    を追加。
  - `src/main/main.ts`
    - `workspace:readFile` IPC handler を追加。
  - `src/main/preload.ts`
    - renderer へ `readWorkspaceFile()` を公開。
- workspace file の表示分類を追加した。
  - `src/main/workspaceService.ts`
    - `.md` は `markdown`
    - `.html/.htm` は `html`
    - `.png/.jpg/.jpeg/.svg/.gif/.webp/.bmp` は `image`
    - `.json/.txt/.csv/.tsv/.xml/.yaml/.yml/.log` と null byte を含まないその他ファイルは `text`
    - それ以外は `unsupported`
    として返すようにした。
    - html は `<base>` を自動注入して相対参照が崩れないようにした。
    - `saveNote()` も `.md` 以外は保存しないようにガードした。
- renderer の tab 表現を markdown 編集 / read-only viewer の union に切り替えた。
  - `src/renderer/App.tsx`
    - open tab state を markdown 専用から generic file tab に拡張。
    - `Ctrl+S` は markdown tab のみ保存するようにした。
    - 既存 tab 再選択時の status message も `編集中/表示中` を出し分けるようにした。
    - placeholder 文言を「ノート」基準から「ファイル」基準へ一部更新した。
- main app 用の file viewer を追加した。
  - `src/renderer/WorkspaceFileViewer.tsx`
    - `html` は iframe
    - `image` は img
    - `text/json/svg など` は text preview
    - 非対応 binary はメッセージ表示
    にした。
  - `src/renderer/styles.css`
    - main tab 高さに追従する viewer スタイルを追加。

## 確認

  - 成功

## 補足

- 今回は main app 側の表示経路だけ対応した。
- file 作成 UI は引き続き markdown note 作成前提のまま。
