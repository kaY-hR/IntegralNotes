# 2026-05-04 15:42 FlexLayout empty group 自動削除

## 依頼

- FlexLayout で複数分割したあと、任意の group の tab をすべて閉じると空 group が残る。
- 空になった group は自動的に閉じたい。

## 対応

- `src/renderer/App.tsx` の FlexLayout model 初期化で `tabSetEnableDeleteWhenEmpty` を `true` にした。
- 初期 tabset の `enableDeleteWhenEmpty` も `true` にし、分割元 group も空になったら削除されるようにした。
- `model.setOnCreateTabSet` を追加し、分割時や全 tab close 後に作られる tabset も空削除対象になるようにした。
- 全 tab を閉じたあとに FlexLayout が再生成する空 tabset には `editor-main` id を付け、既存の tab 追加 fallback と整合するようにした。

## 確認

- `npm run build` 成功。
