# 2026-04-30 10:18 output file 名 suffix 前の underscore 追加

## 目的

解析 block の output slot 初期 file 名を、`slot名A1B.ext` 形式から `slot名_A1B.ext` 形式へ変更する。

## 対応内容

- `src/shared/integral.ts` の既定 output path 生成を、suffix がある場合だけ `_{suffix}` を挟む形に変更した。
- 新規 block 生成、実行前の空 output 補完、output path placeholder は同じ helper を使うため、`/Data/{slot名}_{英数字3文字}{extension}` へ揃う。
- 要求文書と設計文書の命名規則・例を `_A1B` 形式へ更新した。
