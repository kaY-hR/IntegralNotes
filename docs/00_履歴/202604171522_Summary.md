# 2026-04-17 15:22 Summary

## 会話要約

- embed preview のうち `html` は scrollbar 表示も正常だが、`md` だけ scrollbar が出ないという切り分け結果が共有された。
- 原因は readonly Milkdown preview 内で、実際に scroll を持っている `.editor` の親 `.milkdown` に確定高さがなく、`height: 100%` が効き切っていない点だと整理した。

## 実装

- `src/renderer/styles.css`
  - `.readonly-markdown-preview` に `min-height: 0` を追加した。
  - `.readonly-markdown-preview .milkdown` に `height: 100%` を追加した。
  - `.readonly-markdown-preview .milkdown .editor` の `min-height` を `100%` から `0` に変更し、親高さの中で実際に overflow するようにした。

## 確認

- `npm run build` : 通過

## 補足

- 今回の修正は markdown preview 内の scroll container 解決だけに絞っている。
