# 2026-05-04 09:49 F5 Sync shortcut

## 依頼

- F5 キーを押すと Sync が実行されるようにしたい。

## 対応

- `src/renderer/App.tsx` に `runWorkspaceSync` を追加し、既存の Explorer サイドバー `Sync` ボタンと F5 shortcut から同じ処理を呼ぶようにした。
- F5 では Electron / renderer の通常 reload を `preventDefault` / `stopPropagation` で止め、workspace sync 相当の `refreshWorkspace(..., { reloadWorkspaceRuntime: true })` を実行するようにした。
- workspace 未選択時は既存 UI と同じく「ワークスペースフォルダを開いてください。」を表示する。
- Sync 中の重複実行を避けるため、`loadingWorkspace` 中は shortcut / button の実行を抑止し、Sync button は disabled にした。

## 確認

- `npm run build` 成功。
