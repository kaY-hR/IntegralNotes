# 2026-05-08 23:51 .icv viewer の plugin 化

## 背景

`.icv` 画像比較 viewer は当初 `src/renderer/ImageCompareViewer.tsx` として本体に組み込まれていたが、ユーザー意図としては本体機能ではなく viewer plugin として `plugins` 配下へ切り出したい、という整理になった。

## 変更

- `plugins/image-compare-viewer` を追加した。
  - `.icv` 拡張子に対応する `viewers` contribution を持つ。
  - `renderer/index.html` 内で画像切替、重ね表示、layer opacity、drag/drop 順序変更、scale / offset / rotation の一時調整を実装した。
- 本体側の `.icv` 専用 viewer 実装を削除した。
  - `src/renderer/ImageCompareViewer.tsx` を削除。
  - `WorkspaceFileViewKind` から `image-compare` を削除。
  - `WorkspaceFileViewer` の `image-compare` 分岐を削除。
  - `WorkspaceService` の `.icv` hard-code 判定を削除。
  - `styles.css` から本体組み込み viewer 用 CSS を削除。
- viewer plugin から関連 workspace file を読むため、汎用 postMessage bridge を追加した。
  - plugin -> app: `integral:viewer-resolve-workspace-file-url`
  - app -> plugin: `integral:viewer-resolve-workspace-file-url-result`
  - host 側は viewer 対象 file の位置を基準に相対 path を解決し、`resolveWorkspaceFileUrl()` の data URL を返す。
- `package.json` の plugin scripts に `image-compare-viewer` を追加した。
- `src/dev/plugin-installer.mjs` の既定 install 先を開発用 `IntegralNotes-dev` userData に合わせた。
- `docs/30_設計/40_標準描画と結果閲覧.md` と `plugin-sdk/docs/開発者向けアナウンス.md` を更新した。

## 確認

- `npm run plugins:package:all` 成功。
- `npm run plugins:install:all` で `C:\Users\shimadzu\AppData\Roaming\IntegralNotes-dev\plugins\image-compare-viewer` へ install 済み。
- `plugins/image-compare-viewer/renderer/index.html` の inline script を `new Function(...)` で syntax check し、成功。
