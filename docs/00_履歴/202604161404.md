## 実装方針

- まず dataset 実装側から寄せる
  - source dataset は従来どおり visible `.idts`
  - derived dataset は hidden `.store/.integral/datasets/{datasetId}.idts`
  - 実データ directory は `.store/objects/{datasetId}/`
- dataset viewer も source 専用ではなく generic `.idts` viewer に広げる
- 既存の `representation: "directory"` 読み込みは legacy 互換として残しつつ、新規作成は `dataset-json` へ寄せる
