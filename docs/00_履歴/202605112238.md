# 2026-05-11 22:38 shimadzu-lc 静止画像グラジエント編集 UI

## 背景

`shimadzu-lc` の gradient 表示について、折れ線画像自体は固定したまま、折れ線の特徴点に対応する軸メモリ側の値を編集したいという要望があった。

対象値は次の 4 つ。

- 初期濃度
- 初期濃度維持時間
- グラジエント時間
- グラジエント終濃度

## 変更

- `plugins/shimadzu-lc/renderer/index.html`
  - simple mode の `gradient.png` 上に 4 つの小型 input を重ねた。
  - 入力欄は画像に重ねず、画像左側の y 軸目盛り列と画像下側の x 軸目盛り行に配置した。
  - `Initial B.Conc` などの説明ラベルは画面上では隠し、`aria-label` として残した。
  - 入力変更時に `time-prog` と `analysis-time` を更新するようにした。
  - `time-prog` は `0 min 初期濃度 -> 初期濃度維持時間 -> 初期濃度維持時間 + グラジエント時間` の 3 点へ正規化する。
  - 既存 `time-prog` から simple mode 用の 4 値を推定する helper を追加した。

## 確認

- renderer inline script の `new Function(...)` syntax check 成功。
- `npm run build` 成功。
- `npm run install:local` で `%LocalAppData%/IntegralNotes/runtime-plugins/shimadzu-lc` へ再 install 済み。
