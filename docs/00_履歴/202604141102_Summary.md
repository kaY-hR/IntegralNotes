# 2026-04-14 11:02 Summary

## 会話要約

- `data page` とは何か、`data note` と同じなのか、という確認があった。

## 整理した判断

- `data page` は新しい概念として導入したかったわけではなく、`data note` を「ユーザーが preview しながら読むページ」という体験寄りの言い方で説明しただけ。
- 用語を増やすと混乱が増えるため、正式な概念名は既存の `data note` に寄せる方がよい。

## 現時点の推奨整理

- `original data`
  - 実体ファイル / フォルダ
- `data note`
  - original data に 1:1 で対応する Markdown ノート
  - 説明、メタ情報、preview、可視化結果、追記の場
- `data catalog`
  - `data note` や `original data` を一覧・検索・選択するビュー、またはその集合
- `dataset`
  - 解析入出力の内部単位
