## 背景

- chunk viewer に本体側の見出しや card 表現が残っており、`select chunk + viewer` だけの薄い UI にしたいという要望があった。
- あわせて、最近の dialog / block UI が再び角丸 card と pill button に寄っていたため、flat desktop UI 方針を要求文書へ明文化する必要が出た。

## 実装したこと

### flat-UI 方針の明文化

- `docs/10_要求/00_プロダクト概要.md`
  - `GUI はフラットなデスクトップ UI を優先する` を追加
  - VS Code / Obsidian のような UI を基準にし、
    - 大きい角丸 card
    - 強い shadow
    - 強い gradient
    - pill button
    - 装飾のためだけの外枠
    を避ける方針を明記

### chunk viewer の簡素化

- `src/renderer/integralCodeBlockFeature.tsx`
  - `core-display/chunk-view` は共通 preview card を通さず、
    - chunk 選択ボタン
    - renderable viewer
    だけを描画する専用レイアウトへ変更

### 共通スタイルの flatten

- `src/renderer/styles.css`
  - 共通 `button` を小さめ radius の矩形ボタンへ変更
  - dialog card の shadow / 大きい radius を削除
  - plugin card / asset picker / python script dialog / slot row / renderable layout をフラット寄りに調整
  - Integral block の ghost/primary button も pill 表現から外した
