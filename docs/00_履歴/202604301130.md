# 2026-04-30 11:30 @@ promptless continuation

## 目的

Markdown editor 上で `@@` を入力したとき、ユーザーがプロンプトを入力しなくても、AI が現在のノート文脈からよしなに続きを生成して挿入できるようにする。

## 対応内容

- `src/shared/aiChat.ts` に `promptlessContinuation` system prompt と `SubmitPromptlessContinuation*` 型を追加した。
- AI Chat settings dialog に `@@ Promptless continuation` の system prompt 欄を追加し、保存・Reset・空欄チェック対象へ含めた。
- main process に `ai-chat:submitPromptlessContinuation` IPC と `AiChatService.submitPromptlessContinuation` を追加した。
- `@@` 用 agent task では、通常 Markdown 挿入用の `insertMarkdownAtCursor` と、Python block 生成用の `insertPythonBlock` の両方を使えるようにした。
- Python block を選んだ場合は、既存の `>>` と同じく `writeWorkspaceFile` 済み script と `@integral_block` callable を検証してから挿入する。
- `src/renderer/MilkdownEditor.tsx` で `@@` を検出し、marker を削除して即実行するようにした。
- `@@` 実行中はプロンプト入力欄を出さず、進行状態だけを表示する。
- 返却結果が Markdown の場合はそのままカーソル位置へ挿入し、Python block の場合は生成済み Python callable から `itg-notes` block を挿入する。
