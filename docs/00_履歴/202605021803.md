# 2026-05-02 18:03 codebase-review skill 追加

## 依頼

- 既存ソースコードを scan し、肥大化した設計やイマイチなロジックを見直して指摘する skill が欲しい。
- 普遍的に使われている公式 prompt があればコピーしたいが、install は避けたい。

## 対応

- `skill-creator` を使い、repo-local の `.codex/skills/codebase-review` に新規 skill を作成した。
- 外部 install は行わず、OpenAI 公式の Codex review prompt / adversarial review の考え方を確認したうえで、完全コピーではなく要約・汎用化して反映した。
- `SKILL.md` には次を定義した。
  - read-only の既存コードレビューとして実行すること
  - `src/**` を既定 scope とすること
  - 既存 guidance / 履歴 / 要求文書を読んでから判断すること
  - 肥大化、責務混在、async/state の競合、重複、stringly typed contract、境界漏れ、テスト gap などの review 軸
  - findings-first の出力形式
- `agents/openai.yaml` を追加し、UI 表示名・短い説明・default prompt を設定した。
