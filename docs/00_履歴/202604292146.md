# 2026-04-29 21:46 AI Chat tool message の左寄せ

## 目的

AI Chat panel で tool call / tool result のメッセージが中央寄せになっていたため、assistant / LLM 発言と同じ左寄せ表示にする。

- inline AI popup 側は tool message を中央寄せする指定が見当たらなかったため、今回は変更していない。
