# 2026-04-30 11:11 本体設定メニュー追加

## 目的

本体トップのメニューバーに `設定` を追加し、設定ダイアログからデータ登録フォルダとAI関連設定を開けるようにする。

## 対応内容

- トップメニューバーに `設定` ボタンを追加した。
- 本体設定ダイアログを追加し、データ登録時に配置するフォルダを設定できるようにした。
- データ登録フォルダ設定は app `userData/app-settings.json` に保存するようにした。
- 初期値は `\Data` とし、内部では workspace 相対 path に正規化して使うようにした。
- 外部ファイル/フォルダの managed file 取り込み先と dataset manifest 作成先に、設定したフォルダを反映した。
- AI Chat panel の設定ダイアログを `AIChatSettingsDialog` として切り出し、本体設定ダイアログの `AI関連設定` ボタンからも同じUIを開けるようにした。
