# Python 解析 block params 対応の build 確認

## 目的

Python 解析 block の decorator params schema / UI 対応後に build を確認する。

## 確認結果

- 初回の `npm run build` は `vite` が見つからず失敗した。
- `npm install` は 5 分で timeout したが、依存関係は build 可能な状態まで入った。
- 再実行した `npm run build` は成功した。
- Vite CJS Node API deprecated warning と chunk size warning は既存の警告として表示された。
