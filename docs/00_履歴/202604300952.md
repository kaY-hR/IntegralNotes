# 2026-04-30 09:52 AI Chat の明示 skill 呼び出し表示

## 目的

AI Chat panel と inline AI popup で、ユーザーが明示した workspace skill を認識し、添付チップのように表示しつつ LLM 実行にも渡す。

## 対応内容

- `src/shared/aiChat.ts` に skill summary / invocation の型を追加し、chat message と送信 request に skill metadata を持たせた。
- `src/shared/aiChatSkills.ts` を追加し、`$skill-name` / `@skill-name` / 行頭 `/skill-name` / `skill: skill-name` 形式から、利用可能 skill に一致するものだけを抽出するようにした。
- main 側で `.codex/skills` と `Notes/.codex/skills` の `SKILL.md` を列挙し、AI Chat status に `availableSkills` として返すようにした。
- 明示指定された skill は user message に保存し、agent へ渡す user message と system instructions にも反映するようにした。
- AI Chat panel と inline AI popup の入力欄・履歴メッセージに skill chip を表示する `AiSkillChips` を追加した。
- `skill` tool の trace summary を読みやすくするため、tool 入出力の要約を追加した。
