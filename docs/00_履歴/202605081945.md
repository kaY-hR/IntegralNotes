# 2026-05-08 19:45 Explorer パネルの折り畳み操作と sticky parent

## 依頼

- Explorer パネル上部に「全て畳む」アイコンを追加する。
- 大量の file を含む folder を展開してスクロールしたとき、開いている親 folder が上側に隠れないように pin したい。
- pin の挙動は VS Code の sticky scroll のように、現在見ている枝の展開済み祖先 folder を上端に固定するイメージ。

## 対応

- Explorer header actions に「全て畳む」icon button を追加した。
  - `expandedPaths` を空にして全 folder を閉じる。
  - context menu / drop target / inline editor 状態を閉じる。
  - 展開中 folder がない場合は button を disabled にする。
- `FileTree` の展開済み directory row を sticky parent として表示するようにした。
  - 展開済みで children を表示している directory row に `is-sticky-parent` を付与。
  - depth ごとに sticky top をずらし、複数階層の親 folder が上端で重ならないようにした。
  - sticky 表示は CSS の `position: sticky` を使い、各 directory subtree 内だけで効くようにした。

## 確認

- `npm run build` 成功。
