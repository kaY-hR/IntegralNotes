# 2026-04-08 19:15 作業ログ

## 依頼

- `npm run dev` 実行時に renderer 起動で下記エラーが出る
  - `Error [ERR_PACKAGE_PATH_NOT_EXPORTED]: Package subpath './bin/vite.js' is not defined by "exports"`

## 原因

- `src/dev/start-renderer.cjs` が `require.resolve("vite/bin/vite.js")` を使っていた
- Vite 5.4.21 では `vite/bin/vite.js` は package `exports` に公開されておらず、Node の解決時に失敗していた

## 対応

- `src/dev/start-renderer.cjs`
  - `require.resolve("vite/package.json")` で Vite パッケージルートを取得
  - そこから `bin/vite.js` を `path.join(...)` で組み立てて起動する方式に変更

## 検証

- `npm run build` 成功

## メモ

- `npm run dev:renderer` の sandbox 内確認では `esbuild` の子プロセス生成で `spawn EPERM` になった
- これは Codex 側 sandbox 制約の影響で、今回の `ERR_PACKAGE_PATH_NOT_EXPORTED` とは別問題
