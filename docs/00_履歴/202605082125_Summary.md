# 2026-05-08 21:25 画像比較 view plugin 初版

## 依頼

- 位置合わせ済み画像セットを比較するための viewer を実装する。
- 複数画像を bundle した JSON file を定義し、その拡張子専用の view plugin として扱う。
- 左右キーで画像切替、画像ごとの透過率、重ね順 drag-drop、サイズ / offset / 回転調整を可能にする。

## 対応

- `.icv` を image compare manifest として追加した。
  - `WorkspaceFileViewKind` に `image-compare` を追加。
  - `WorkspaceService` で `.icv` を UTF-8 JSON として読み、専用 kind を返すようにした。
- `src/renderer/ImageCompareViewer.tsx` を追加した。
  - `images` / `layers` / `members` 配列を受け付ける。
  - 各 layer は `path` / `relativePath` / `src` / `file`、`id`、`label`、`opacity`、`scale`、`offsetX`、`offsetY`、`rotation` を扱える。
  - manifest からの相対 path を workspace path として解決する。
  - 左右キーと前後 button で active image を切り替える。
  - `切替` / `重ね` 表示を切り替えられる。
  - layer ごとの透過率 slider、drag-drop による上下入れ替え、active layer の scale / offset / rotation 調整を追加した。
- `WorkspaceFileViewer` に `image-compare` 分岐を追加した。
- `styles.css` に画像比較 viewer 用のフラットな UI style と狭幅対応を追加した。
- `docs/30_設計/40_標準描画と結果閲覧.md` に `.icv` manifest 仕様を追記した。

## 確認

- `npm run build` 成功。

## 注意

- 初版は workspace path ベースで画像を解決する。managed data ID からの直接解決は未実装。
- `.idts` は従来どおり data-note entrypoint、`.icv` は file tab 上の画像比較 viewer として扱う。
