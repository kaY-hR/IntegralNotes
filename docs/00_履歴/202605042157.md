# 2026-05-04 21:57 FlexLayout tab shortcuts

## 依頼

- FlexLayout の tab を `Ctrl+Tab` で切り替えたい。
- `Ctrl+W` で選択中 tab を閉じたい。
- 実装が煩雑なら必須機能ではないため見送ってもよい。

## 対応

- `src/renderer/App.tsx` に FlexLayout の選択中 tabset 内で隣接 tab を探す helper を追加した。
- `Ctrl+Tab` で次 tab、`Ctrl+Shift+Tab` で前 tab を選択する shortcut を追加した。
- `Ctrl+W` で選択中 tab を閉じる shortcut を追加した。
- Markdown tab は既存の `requestCloseDirtyTab` 経由で閉じ、未保存時の確認を維持した。
- workspace tool tab は FlexLayout tab として直接閉じるようにした。

## 確認

- `npm run build` 成功。
