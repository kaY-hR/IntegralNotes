## 相談内容の整理

- ユーザーから、VSCode / Obsidian のように、ファイルやフォルダの新規作成・リネームをツリー内で直接編集したいという要望が出た
- 現行実装では、作成・リネーム・削除の本体処理は `src/renderer/App.tsx` にまとまっており、現在はその入力 UI が `WorkspaceDialog` になっているだけ
- そのため、`create` と `rename` を inline input に差し替えるのは十分現実的
- 実装方針としては、まず `create` / `rename` を inline 化し、`delete` の確認ダイアログだけは当面維持するのが安全
