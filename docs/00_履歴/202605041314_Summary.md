# Relation Graph runtime error 修正

## 発生した問題

`npm run dev` で Relation Graph を開くと、renderer 側で以下の runtime error が発生した。

```text
InvalidArgumentsGraphError: DirectedGraph.from: inconsistent indication that the graph should be multi in given options!
```

原因は `src/renderer/RelationGraphView.tsx` で `DirectedGraph` に `multi: true` を渡していたこと。graphology の `DirectedGraph` は multi graph 指定と矛盾するため、同一 source / target 間に複数 edge を持たせる場合は `MultiDirectedGraph` を使う必要がある。

## 修正

- `RelationGraphView.tsx` の import を `DirectedGraph` から `MultiDirectedGraph` に変更した。
- graph instance 作成時の `multi: true` を削除し、`new MultiDirectedGraph({ allowSelfLoops: false })` にした。
- 同一 source / target 間で複数 directed edge を追加できることを node one-liner で確認した。

## 検証

- `npm run build` 成功。
- `node -e` で `MultiDirectedGraph` に同一 source / target の edge を 2 本追加できることを確認した。
