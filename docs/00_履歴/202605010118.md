# 2026-05-01 01:18 npm warning / vulnerability 対応

## 背景

- npm 実行時の warning と vulnerability が気になっていた。
- 直近履歴では `npm run build` に Vite CJS API deprecated warning と chunk size warning が残っていた。

## 対応

- `npm audit fix` で semver 範囲内の transitive dependency を更新した。
  - `@xmldom/xmldom`
  - `axios`
  - `follow-redirects`
  - `dompurify`
  - `postcss`
- direct dev dependency を major 更新した。
  - `electron`: `^35.7.5` -> `^41.3.0`
  - `vite`: `^5.4.14` -> `^8.0.10`
  - `@vitejs/plugin-react`: `^4.3.4` -> `^6.0.1`
- Vite build の chunk size warning は、Electron desktop renderer として現在の bundle size を許容する前提で `chunkSizeWarningLimit: 2000` を明示した。
- dev 実行時に Vue esm-bundler feature flag warning が出たため、Vite の `define` で Vue feature flags を明示した。

## 確認

- `npm audit --json`
  - vulnerabilities total: `0`
- `npm run build`
  - 成功。
  - Vite CJS API deprecated warning は解消。
  - chunk size warning も解消。
- `npm run dev`
  - renderer / main watcher は起動し、TypeScript error は 0。
  - Vue feature flag warning を確認し、追加対応した。
- `node node_modules\electron\cli.js --version`
  - `v41.3.0`
- Vite dev server short check
  - `http://127.0.0.1:5174/` で応答確認。
