## 背景

- `データ登録 -> source chunk 作成` の blob 選択 UI で、checkbox を入れても直後に外れる不具合があった。
- あわせて、blob 一覧の主表示が `blobId` 側に寄っており、元ファイル名を主表示にしたいという要望があった。
- blob artifact note についても、見出しを `ファイル名_blob-id` にした方が内容把握しやすいと判断した。

## 実装したこと

### checkbox リセット不具合修正

- `src/renderer/IntegralAssetDialogs.tsx`
  - `BlobSelectionDialog` の `useEffect` が `initialSelectedBlobIds` の参照変化で毎回走っていたため、未指定時に空配列で選択状態を上書きしていた
  - 依存を安定化し、初期選択 seed が変わったときだけ再初期化するよう修正

### blob 表示順の見直し

- `src/renderer/IntegralAssetDialogs.tsx`
  - blob 一覧の主ラベルを `originalName`
  - 補助表示を `blobId`, `sourceKind`
  に変更

### blob artifact note 見出し

- `src/main/integralWorkspaceService.ts`
  - 新規作成される blob artifact note のタイトルを `# {originalName}_{blobId}` に変更
  - 本文にも `Blob ID` 行を追加

## 補足

- `npm run build` は成功
- artifact note のタイトル変更は新規作成分から反映される
