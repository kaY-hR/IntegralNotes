# 2026-05-11 23:29 shimadzu-lc 余白調整と image block エラー修正

## 依頼

- `npm run dev` 中に `MilkdownError: Context "imageBlockConfigCtx" not found` が出た。
- `shimadzu-lc` の runtime block 表示で余白が多い。

## 対応

- `src/renderer/workspaceEmbedFeature.tsx`
  - workspace embed でない通常画像を Milkdown 標準 image block view へ直接委譲しないよう変更。
  - 軽量な標準画像 node view を追加し、`imageBlockConfigCtx` 未注入時の runtime error を回避。
  - workspace 画像 path の proxy 解決を維持するため `proxyImageUrl` を受け取るようにした。
- `src/renderer/MilkdownEditor.tsx`
  - workspace embed feature に `proxyImageUrl` を渡すよう変更。
- `src/renderer/integralCodeBlockFeature.tsx`
  - custom renderer 付き external plugin block では説明文を非表示にして縦幅を削減。
- `src/renderer/styles.css`
  - plugin iframe の既定高さを `560px` から `430px` に短縮。
  - 標準画像 node view 用の最低限の CSS を追加。
- `plugins/shimadzu-lc/renderer/index.html`
  - renderer 内の `height:100%` 前提の伸びと余白を抑制。
  - iframe 幅 860px 以下で古い mobile padding が戻らないよう調整。

## 確認

- `plugins/shimadzu-lc/renderer/index.html` の script 構文チェック成功。
- `npm run build` 成功。
- `plugins/shimadzu-lc` で `npm run install:local` 成功。
