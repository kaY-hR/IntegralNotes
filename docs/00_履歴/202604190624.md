## 概要

- `share_note_with_input` 導入後の整理として、output の data-note 投影設定を「共有先 note へ埋め込むかどうか」の真偽値で指定できるようにした。
- 既存の `project_to_inputs` は後方互換として残しつつ、`embed_to_shared_note = true` が指定された場合は `share_note_with_input` の共有先を優先して投影する実装に変更した。

## 実装

- `src/shared/integral.ts`
  - `IntegralSlotDefinition` に `embedToSharedNote?: boolean` を追加。
- `src/main/integralWorkspaceService.ts`
  - `appendProjectedOutputsToDataNotes()` で参照する投影先解決を `resolveProjectedInputSlotNames()` に分離。
  - `embedToSharedNote = true` かつ `shareNoteWithInput` がある場合は、その input slot のみを投影先とする。
  - 上記に該当しない場合は既存どおり `projectToInputs` を利用する。
  - Python slot parser で `embed_to_shared_note` を bool として解釈し、slot 定義へ反映する。
