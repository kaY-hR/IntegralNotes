# 2026-04-08 18:28 作業ログ

## 依頼

- `npm run dev` 再実行時に Electron ウィンドウが 2 個出る
- ログには cache access denied / gpu cache failed が出ている

## 原因

- 既存の Electron インスタンスが残っている状態で、次の `npm run dev` がもう1つ Electron を起動していた
- 同一アプリの複数インスタンスが同じ user data / cache を触ろうとして、Windows 側で cache 関連エラーが出ていた

## 対応

- `src/main/main.ts`
  - `app.requestSingleInstanceLock()` を追加
  - 2個目の起動要求では新規ウィンドウを作らず、既存ウィンドウを restore/focus するように変更
  - lock を取れなかったプロセスは即 `app.quit()` するように変更

## 検証

- `npm run build` 成功

## メモ

- この修正は「修正後の Electron main」が起動してから有効
- そのため、一度古い dev ウィンドウ/プロセスを閉じてから `npm run dev` をやり直す必要がある
