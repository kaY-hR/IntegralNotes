## 実施内容

- `src/main/main.ts`
  - `Ctrl/Cmd + -`, `Ctrl/Cmd + +`, `Ctrl/Cmd + 0` を明示的に処理する zoom handler を追加。
  - `zoom-changed` を拾って、ホイールズーム/ピンチ相当の zoom 要求も同じ経路で処理するようにした。
  - `VITE_DEV_SERVER_URL` が無いときだけ `mainWindow.removeMenu()` を呼び、リリース版で Electron デフォルトメニューを非表示にした。
- `src/renderer/App.tsx`
  - 初回ロード時だけ「最初のノートを自動オープン」する ref を追加し、最後のタブを閉じた後に勝手に再オープンしないようにした。
  - エクスプローラー空白クリック時に選択解除する処理を追加。
  - エクスプローラー空白右クリック時に root 用コンテキストメニューを開き、root 直下へ `New Note` / `New Folder` を作成できるようにした。
  - tree item 用コンテキストメニューを `entry/root` の両対応に整理し、root メニューでは rename/delete を出さないようにした。

## 効果

- `Ctrl + -` 後でも `Ctrl + +` で拡大に戻せる。
- 既存項目を外して root 直下作成に入りやすくなった。
- タブを全部閉じたときに直前ノートが勝手に戻る挙動を止めた。
- インストール版では Electron 既定メニューを隠せる。
