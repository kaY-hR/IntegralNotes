## 背景

- plugin runtime を `external only` に寄せる方針で進めることになった。
- 最終形は plugin store + GUI install、開発中は zip 配布 + bat で回す方針を整理した。
- ユーザーに Node.js を要求しない install 導線が必要になった。

## 判断

- runtime は `external plugin` だけを見ればよい構成に揃えた。
- repo 内の `plugins/*` は runtime install 先ではなく source と sample 実装の置き場にした。
- エンドユーザー向けには Node.js を要求せず、現在は zip + bat、次段階で GUI import / store に寄せる方針とした。
