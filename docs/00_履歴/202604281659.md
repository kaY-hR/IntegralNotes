# 2026-04-28 会話まとめ

## 目的

AI Chat panel の Settings dialog から、通常 chat、`??` の inline AI insertion、`>>` の Python block implementation で使う system prompt を確認・編集できるようにする。

## 実装内容

- `src/shared/aiChat.ts` に `AiChatSystemPrompts` と既定 system prompt を追加した。
- AI Chat status に現在の system prompt と既定値を含め、settings 保存 request で prompt 群を保存できるようにした。
- `src/main/aiChatService.ts` で prompt 設定を `ai-chat-settings.json` に永続化し、未設定・空文字の場合は既定 prompt へ戻すようにした。
- 通常 AI Chat の system prompt を `src/main/aiAgentService.ts` に渡せるようにし、設定値を反映するようにした。
- `??` の inline Markdown insertion と `>>` の Python block implementation の system prompt を、設定値から組み立てるようにした。
- `src/renderer/AIChatPanel.tsx` の Settings dialog に System Prompts セクションを追加し、3種類の prompt を textarea で確認・編集・既定値リセットできるようにした。
- `src/renderer/styles.css` に prompt editor 用の幅・textarea 表示を追加した。
