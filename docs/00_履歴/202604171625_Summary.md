- `src/renderer/ExternalPluginFileViewer.tsx`
  - plugin viewer iframe に一時的に追加していた `scrolling=\"yes\"` を削除した。

  - scrollbar 可視化のために足していた `scrollbar-gutter: stable` を削除した。

    `editor-surface .readonly-markdown-preview ...` の override は残した。
