## 会話概要

- ユーザー要望: `.idts` file を main app の built-in viewer で開けるようにしたい
- 表示要件:
  - データファイル名
  - リンク
  - ノートの埋め込み表示
- 方針:
  - plugin viewer ではなく built-in viewer として実装
  - 既存 plugin には影響させない
  - `.idts` は source dataset manifest として解釈し、`memberIds` と dataset note をまとめて表示

## 実装内容

- `src/shared/workspace.ts`
  - `dataset-json` viewer kind を追加
  - `.idts` viewer 用の manifest view model を追加
- `src/main/integralWorkspaceService.ts`
  - `.idts` manifest を読んで `memberIds` から元データ metadata を解決する special document reader を追加
