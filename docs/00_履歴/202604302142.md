# 2026-04-30 21:42 解析 block input picker preview

## 背景

- 解析 block の input picker で候補を選ぶ前に中身を確認したい。
- image / html など app 内で描画可能な managed file は実体を preview し、非対応 file は data-note を preview したい。
- dataset は dataset 自体の data-note と、構成 managed file の data-note を同じ preview window 内で並べて見たい。

## 対応

- `src/renderer/IntegralAssetDialogs.tsx` に `IntegralAssetPreviewWindow` を追加した。
  - managed file が `image` / `html` / plugin viewer 対応なら実体を表示する。
  - それ以外の managed file は data-note を Markdown preview で表示する。
  - data-note がない Markdown / text file は file 内容を fallback preview する。
  - dataset は dataset note と member note を flex layout で並べる。
  - preview window は header drag で移動、右下 handle で resize できる。
- `src/renderer/integralCodeBlockFeature.tsx` の inline input picker option に preview target を持たせ、hover / focus で floating preview を表示するようにした。
- `src/renderer/styles.css` に preview window / dataset note layout / resize handle の style を追加した。

## 確認

- `npm run build`
  - 成功。
  - Vite の CJS API deprecated warning と chunk size warning は既存系の警告として残る。
