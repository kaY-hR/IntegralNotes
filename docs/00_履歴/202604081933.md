# 2026-04-08 19:33 作業ログ

## 依頼

- サイドバーも上部メニューも見えない
- 見直したところ、追加 UI が `FlexLayout` の後ろに入っているように見える

## 原因

- `flexlayout-react` の `.flexlayout__layout` は `position: absolute` で描画される
- それに対して `src/renderer/styles.css` の `.workspace__layout` に `position: relative` が入っていなかった
- その結果、FlexLayout のルートが本来の編集領域だけでなく、上部メニューやサイドバーを含む広い領域に被さって表示されていた

## 対応

- `src/renderer/styles.css`
  - `.workspace__layout` に `position: relative` を追加
  - これで FlexLayout の absolute 配置が編集領域内に閉じるように修正

## 検証

- `npm run build` 成功

- 既に `npm run dev` を起動中なら、watch で CSS は即反映される想定
- 反映が怪しければ Electron ウィンドウを閉じて `npm run dev` をやり直す
