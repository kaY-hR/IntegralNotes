# 2026-04-29 会話まとめ

## 目的

AI Chat panel と inline AI popup の応答を、完了後一括表示ではなく live streaming 表示にする。

## 実装内容

- `ToolLoopAgent.generate()` 経路に加えて、streaming 用に `ToolLoopAgent.stream()` を使う経路を追加した。
- main process から renderer へ `ai-chat:streamEvent` IPC を追加し、text delta / text reset / step-level tool trace / started / finished を送れるようにした。
- AI Chat panel では送信中の pending assistant message に text delta を追記表示し、tool loop の step 完了時には live tool trace を表示するようにした。
- inline AI popup (`??` AI 挿入、`>>` Python block 生成) でも送信中の assistant streaming 表示と live tool trace 表示を追加した。
- final result と履歴保存は従来通り、IPC invoke の戻り値で確定した assistant message / tool message を使う構成を維持した。
- 要求文書と AI Chat tooling 設計文書から「streaming 未実装」の記述を更新した。
