## 会話概要

- ユーザー要望:
  - viewer plugin を `![]()` の埋め込み表示でも使えるようにしたい
  - その内容を Issue として切りつつ、そのまま実装したい
- 方針:
  - embed でも file tab と同じ viewer registry を使う
  - 初版は preview-only とし、embed surface の click / Enter / Space で通常の file tab を開く
  - plugin viewer には `presentation: "embed" | "full"` を渡し、必要なら UI 密度を変えられるようにする
