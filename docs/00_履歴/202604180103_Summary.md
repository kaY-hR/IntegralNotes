# Summary

- Python 実行 block の output 名入力欄の枠線が、`1px` にしても見た目に反映されない問題を調査した。
- ビルド済み CSS を確認したところ、Milkdown 側に `.milkdown input { border: none; background: none; box-shadow: none; }` という広い reset が含まれていることを確認した。
- `src/renderer/styles.css` の `.integral-output-slot-row__name-input` について、`.editor-surface .milkdown ...` を含むより強い selector に変更し、`border`, `background`, `color`, `box-shadow`, `padding` を `!important` 付きで上書きするよう修正した。
- あわせて `appearance: none` と `-webkit-appearance: none` を追加し、input 固有の見た目が残らないようにした。
- `npm run build` を実行し、ビルド成功を確認した。
