# 2026-05-04 09:52 Block / Embed 折り畳み UI

## 依頼

- Python などの Integral block に折り畳み機能を付けたい。
- `![]()` などの workspace 埋め込みにも同じ折り畳み機能を付けたい。
- 実行済みと判定された Python block は自動で折り畳みたい。

## 対応

- `src/renderer/CollapseToggleButton.tsx` を追加し、block と embed で共通のアイコン型折り畳み button を使うようにした。
- `src/renderer/integralCodeBlockFeature.tsx` で Integral block header に折り畳み button を追加した。
- 実行済み block 判定が true になった block は、その output 状態ごとに一度だけ自動折り畳みするようにした。
- 手動で展開した実行済み block は、同じ output 状態のままなら再描画で勝手に折り畳まれないようにした。
- `src/renderer/workspaceEmbedFeature.tsx` で workspace embed toolbar の左側に折り畳み button を追加し、畳んだときは preview surface と resize handle を隠すようにした。
- `src/renderer/styles.css` に折り畳み button、collapsed block、collapsed embed の style を追加した。

## 確認

- `npm run build` 成功。
