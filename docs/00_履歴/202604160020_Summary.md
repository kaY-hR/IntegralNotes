## 会話要約

- 直前までで workspace embed 自体は動くようになっていたが、仕上げとして
  - file 名や拡張子表示は不要
  - `md` は plain text ではなく Milkdown の rich preview にしたい
  - embed block を click したら別 tab 表示したい
  という追加要望が出た。

## 実装

- `src/renderer/workspaceEmbedFeature.tsx`
  - embed renderer の header / badge 表示を外し、preview 本体だけを描画する構成へ整理した。
  - `.md` と managed `data-note` fallback は iframe text preview ではなく readonly の `Crepe` を使った markdown preview に切り替えた。
  - embed が workspace file を指す場合は `OPEN_WORKSPACE_FILE_EVENT`、managed `data-note` fallback の場合は `OPEN_MANAGED_DATA_NOTE_EVENT` を発火し、block 全体 click で対応 tab を開けるようにした。
  - markdown embed preview 内では workspace image path を DOM 用 URL に解決して表示できるようにした。
- `src/renderer/styles.css`
  - embed card の padding / margin / radius を縮め、filename や badge のないコンパクトな preview card に寄せた。
  - click hitbox と markdown preview 用の style を追加し、embed 内の readonly Milkdown を小さく収める見た目に調整した。
- `docs/Issues.md`
  - `Issue 26` に今回の仕上げ内容を追記した。
