# 2026-04-21 14:30 AI Chat binary file discovery fix

## 背景

- AI Chat の `bash-tool` で `find Data` を実行した際、`Notes/Data` 配下に実在する `.png` が列挙されなかった。
- 原因は Windows fallback の workspace preload が `text file only` だったため。binary / image / oversized file を snapshot に積んでおらず、`find` / `ls` からも見えなかった。

## 実装

- `src/main/aiAgentService.ts`
  - workspace snapshot 作成処理を更新。
  - text file は従来どおり実コンテンツを preload。
  - binary / image / oversized file は placeholder text を持つ stub file として snapshot に追加。
  - これにより `find` / `ls` では画像ファイルも discover 可能になった。
  - bash 向け instructions も更新し、text は実内容、binary / oversized は placeholder stub の可能性があることを明記。

## 確認

- `npm run build` 成功。
- `Notes/Data` の実ファイル集計では `.png` が 2 件存在することを確認済み。

## 判断メモ

- current npm latest の `just-bash 2.14.2` / `bash-tool 1.3.16` では Windows `OverlayFs` 問題が残っているため、現時点では in-memory snapshot fallback を継続。
- 今回の修正は、その fallback 上でも binary file discovery を失わないためのもの。
