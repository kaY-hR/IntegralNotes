# Relation Graph の 1hop 強調とドラッグ物理挙動

## 要望

- Relation Graph のアクティブタブ近傍の強調表示を default 1hop にする。
- ノードをドラッグしたとき、周辺ノードがばねのように追従して動くようにする。
- 配置状態の永続化は不要。

## 実装

- `src/renderer/RelationGraphView.tsx`
  - `ACTIVE_NEIGHBORHOOD_HOPS` を `1` に変更。
  - sigma の `downNode` と mouse captor の `mousemovebody` / `mouseup` / `mouseleave` を使い、ノードドラッグを追加。
  - ドラッグ中は camera panning を一時停止し、ノード座標を graphology 側で更新。
  - 表示中の edge をばねとして扱い、ドラッグ中とリリース後に短時間だけ物理更新するようにした。
  - ドラッグ後に `clickNode` が発火してノートを開かないよう、移動量が一定以上の場合だけクリックを短時間抑制。
  - 大きいグラフでは負荷を抑えるため、ばね物理更新は 400 nodes までに制限。
- `src/renderer/styles.css`
  - Relation Graph canvas 領域に `grab` / `grabbing` cursor を追加。

## 確認

- `npm run build` 成功。
