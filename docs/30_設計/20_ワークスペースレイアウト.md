# ワークスペースレイアウト

## 全体

```text
Workspace/
  Notes/
  Artifacts/
  blob/
  .chunk/
  .py-scripts/
```

## Notes

- 通常ノート
- `itg-notes` block を含む Markdown

## Artifacts

ユーザーがノートから参照する Markdown facade。

例:

```text
Artifacts/
  BLB-7K2M9Q4D.md
  CNK-9X4Q2M1A.md
```

方針:

- blob artifact は自動作成
- chunk artifact は必要時のみ作成

## blob

```text
blob/
  BLB-7K2M9Q4D/
    blob.json
    payload/
      ...
```

`blob.json` 例:

```json
{
  "blobId": "BLB-7K2M9Q4D",
  "originalName": "sample01.lcd",
  "createdAt": "2026-04-13T10:00:00+09:00"
}
```

## .chunk

```text
.chunk/
  CNK-1A2B3C4D/
    chunk.json
    links.json
  CNK-9X4Q2M1A/
    chunk.json
    result.html
    table.csv
```

### source chunk

```text
.chunk/
  CNK-1A2B3C4D/
    chunk.json
    links.json
```

`chunk.json` 例:

```json
{
  "chunkId": "CNK-1A2B3C4D",
  "kind": "source-bundle",
  "createdByBlockId": null,
  "createdAt": "2026-04-13T10:05:00+09:00"
}
```

`links.json` 例:

```json
{
  "members": [
    {
      "blobId": "BLB-7K2M9Q4D",
      "target": "../../blob/BLB-7K2M9Q4D/payload"
    },
    {
      "blobId": "BLB-9X4Q2M1A",
      "target": "../../blob/BLB-9X4Q2M1A/payload"
    }
  ]
}
```

### derived chunk

```text
.chunk/
  CNK-9X4Q2M1A/
    chunk.json
    result.html
    preview.png
```

`chunk.json` 例:

```json
{
  "chunkId": "CNK-9X4Q2M1A",
  "kind": "PYS-7K2M9Q4D.result",
  "createdByBlockId": "BLK-1F8E2D0A",
  "createdAt": "2026-04-13T10:10:00+09:00"
}
```

## .py-scripts

```text
.py-scripts/
  PYS-7K2M9Q4D/
    script.json
    pca.py
    helper.py
    config.json
    analysis-args.json
    stdout.log
    stderr.log
```

### ルール

- 実行場所は `PYS-...` フォルダ自身
- entry ファイル名は元のまま保持する
- 同梱ファイルはフラット配置する
- `analysis-args.json`, `stdout.log`, `stderr.log` は上書きされてよい

## GC 対象

MVP では、次を GC 対象にできる。

- どこからも参照されない chunk

一方、次は GC 対象にしない。

- blob
- `.py-scripts/PYS-...`
- artifact note
