# ワークスペースレイアウト

## 全体

```text
Workspace/
  Notes/
  data-catalog/
  .original-data/
  dataset/
  .py-scripts/
```

## Notes

- 通常ノート
- `itg-notes` block を含む Markdown

## data-catalog

ユーザーがノートから参照する Markdown facade。

例:

```text
data-catalog/
  sample01.lcd_BLB-7K2M9Q4D.md
  CNK-9X4Q2M1A.md
```

方針:

- original data の data note は自動作成
- dataset の data note は必要時のみ作成

## .original-data

```text
.original-data/
  BLB-7K2M9Q4D/
    original-data.json
    payload/
      ...
```

`original-data.json` 例:

```json
{
  "originalDataId": "BLB-7K2M9Q4D",
  "originalName": "sample01.lcd",
  "createdAt": "2026-04-13T10:00:00+09:00"
}
```

## dataset

```text
dataset/
  CNK-1A2B3C4D/
    dataset.json
    links.json
  CNK-9X4Q2M1A/
    dataset.json
    result.html
    table.csv
```

### source dataset

```text
dataset/
  CNK-1A2B3C4D/
    dataset.json
    links.json
```

`dataset.json` 例:

```json
{
  "datasetId": "CNK-1A2B3C4D",
  "kind": "source-bundle",
  "createdByBlockId": null,
  "createdAt": "2026-04-13T10:05:00+09:00"
}
```

`links.json` 例:

```json
{
  "members": [
    {
      "originalDataId": "BLB-7K2M9Q4D",
      "target": "../../.original-data/BLB-7K2M9Q4D/payload"
    },
    {
      "originalDataId": "BLB-9X4Q2M1A",
      "target": "../../.original-data/BLB-9X4Q2M1A/payload"
    }
  ]
}
```

### derived dataset

```text
dataset/
  CNK-9X4Q2M1A/
    dataset.json
    result.html
    preview.png
```

`dataset.json` 例:

```json
{
  "datasetId": "CNK-9X4Q2M1A",
  "kind": "PYS-7K2M9Q4D.result",
  "createdByBlockId": "BLK-1F8E2D0A",
  "createdAt": "2026-04-13T10:10:00+09:00"
}
```

## .py-scripts

```text
.py-scripts/
  PYS-7K2M9Q4D/
    script.json
    pca.py
    helper.py
    config.json
    analysis-args.json
    stdout.log
    stderr.log
```

### ルール

- 実行場所は `PYS-...` フォルダ自身
- entry ファイル名は元のまま保持する
- 同梱ファイルはフラット配置する
- `analysis-args.json`, `stdout.log`, `stderr.log` は上書きされてよい

## GC 対象

MVP では、次を GC 対象にできる。

- どこからも参照されない dataset

一方、次は GC 対象にしない。

- original data
- `.py-scripts/PYS-...`
- data note
