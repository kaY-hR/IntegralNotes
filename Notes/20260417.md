`Data/LC_text_files` にデータファイルを配置した。

<br />

(まずはこれらを右クリックして、データセット化した。)

(ここで "\["を押して、"my\_LC\_Samples.idtsを選ぶとリンクが作れる")

[my\_LC\_Samples.idts](/datasets/my_LC_Samples.idts)

<br />

データセットをjsonに変換する。

（ここで">"を押して、LC text to Chromatogram JSONを選ぶと、下記のブロックが挿入される）

<br />

```itg-notes
id: BLK-24C140CB
run: src/lc_text_to_chromatogram_json.py:main
in:
  source: /Data/my-LC-Samples.idts
params: {}
out:
  json:
    dir: /Data
    name: my-LC-Samples
    latest: /Data/my-LC-Samples.json
```

`//そもそもユーザーが中間ファイル(json)を意識するのはイケてなさすぎる`

<br />

正しく変換できているか確認のため、可視化。

（ここで">"を打って、Chromatogram JSON To Plotly HTMLを選ぶと、下記のブロックが挿入される）

<br />

`//読み込んだ時点で可視化すべき。PCAとか数値処理のためにjsonは必要だが、jsonと一緒に可視化プロットまで出しても良い。`

```itg-notes
id: BLK-D67E7890
run: src/chromatogram_json_to_plotly_html.py:main
in:
  source: /Data/my-LC-Samples.json
params: {}
out:
  plot:
    dir: /Data
    name: plot
    latest: /Data/plot.html
```

<br />

(ここで、"!\[" と打って\`クロマトのグラフ\`を選ぶと、下記のブロックが挿入される)

![1.00](/Data/plot.html)

４本のピークがあるクロマトグラムである。

`//見切れていて見にくい`

<br />

次に、PCAを実行

（ここで、">"を打って、Chromatogram PCAを選択し、Sourceにjsonのデータを選ぶ）

`//jsonのデータを選ぶのは非直観的`

```itg-notes
id: BLK-B36CFD3C
run: src/chromatogram_pca.py:main
in:
  source: /Data/my-LC-Samples.json
params: {}
out:
  report:
    dir: /Data
    name: PCA-result
    latest: /Data/PCA-result.idts
```
