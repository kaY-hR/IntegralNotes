#### データの管理について

<br />

各データには、.store/.integral/ORD-04E01C3C.json の様に、ORDから始まるIDを付けて管理しています。

一応、ファイル内容 or ファイルパスのいずれか片方のみが変化するだけなら、自動で追跡します。

<br />

#### データセットの概念

<br />

pythonの実行については、データセットという概念を単位にしています。

いくつかのデータをエクスプローラパネルで複数選択して、右クリックからデータセットを作ることが出来ます。

<br />

実体としては、下記の様なjsonで管理しています。

[DTS-64F0B897.json](/.store/.integral/DTS-64F0B897.json)

<br />

pythonの実行については、ファイルが1つでも複数でも、基本的にデータセットとして出力します。

データセットを確認したい場合は、\`!\[\`と打って、DTS-..から始まるファイルを選択すれば埋め込み表示で確認できます。
